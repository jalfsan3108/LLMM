Encabezado Principal

Puedes usar texto en cursiva, o aplicarle subrayado. También es útil el texto tachado para indicar elementos eliminados.

Encabezado Secundario

Aquí se usan distintos estilos como la negrita y el texto resaltado para destacar información clave. Además, puedes insertar subíndices y sobrescritos, lo cual es útil para fórmulas matemáticas o notas.

Subtítulo o Encabezado Terciario

Este párrafo contiene una cita importante:
“La simplicidad es la máxima sofisticación.” — Leonardo da Vinci
Además, puedes incluir código en línea como este ejemplo: <h1>Hola Mundo</h1>.

En este párrafo, usamos una abreviatura con una explicación sobre ella: ONU. También se incluyen fracciones en texto: 1er lugar, 2do lugar.
